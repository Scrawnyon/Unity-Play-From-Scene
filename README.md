# Unity Play From Scene
A very simple solution for a very simple (but common) problem. Uses Unity's EditorSceneManager.playModeStartScene

## Usage
Open the scene you want to start from. Click "Editor Start Scene > Set Current As Start" to set the scene, and "Clear" to return to default behaviour
